shutej.github.io
================

To clone:

    git clone --recursive git@github.com:shutej/shutej.github.io.git

To preview the build, use:

    hugo serve -t purehugo

To build, use:

    hugo -t purehugo

To push:

    git subtree push --prefix=public git@github.com:shutej/shutej.github.io.git master
